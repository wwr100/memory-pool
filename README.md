项目名称
C++ 高并发内存池

项目地址
https://github.com/wwr100/memory-pool.git

项目描述
基于 C++ 实现的一个线程安全的高并发内存池，旨在优化内存分配和释放的性能，特别是在多线程环境下。项目包括内存池的设计与实现、性能优化以及测试验证。

项目实现
设计三层缓存架构：实现了 ThreadCache、CentralCache 和 PageCache 三层缓存架构，通过分层管理提升内存分配效率，减少锁竞争，优化多线程访问性能。

实现高效同步机制：采用 thread_local、自旋锁和原子操作等多种同步机制，在不同层级选择最优的同步策略，显著提升并发性能。
